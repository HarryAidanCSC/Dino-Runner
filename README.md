# Dino-Runner

## Overview
This project uses image recognition and automated workflows in python to play the [Chrome Dino](chrome://dino/) game.

## How to install
Ensure you have Python and [uv](https://github.com/astral-sh/uv) installed.

1. Clone from GitHub (SSH)
```bash
git clone git@github.com:HarryAidanCSC/Dino-Runner.git
```

2. Enter the project directory
```bash
cd Dino-Runner
```

## How to run

```bash
uv sync
uv run src/main.py
```
